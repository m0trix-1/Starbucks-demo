# Starbucks Clone - Web Build

Dự án giao diện Starbucks Premium phục vụ cho việc thực hành Front-end và thiết kế Responsive.

## Cấu trúc thư mục
Đã được cấu trúc theo chuẩn HTML/CSS/JS thuần, dễ dàng deploy lên các nền tảng static hosting.

## Deploy lên GitHub Pages
1. Khởi tạo repository trên GitHub và push toàn bộ source code này lên.
2. Vào **Settings** > **Pages**.
3. Tại phần **Source**, chọn nhánh `main` (hoặc `master`) và thư mục `/(root)`.
4. Nhấn Save. Đợi 1-2 phút, GitHub sẽ cấp cho bạn một đường link public để xem website.

## Yêu cầu tài nguyên
Vui lòng upload các ảnh sản phẩm thực tế vào thư mục `assets/img/` để UI hiển thị chính xác nhất.
